This project is a **Power BI dashboard** designed to analyze **country-wise sales performance** using interactive visualizations and DAX measures.  
The goal of this project is to demonstrate **data analysis, visualization, and business insight generation** skills.



# Project Overview

The dashboard helps answer key business questions such as:
- Which countries generate the highest sales?
- How does sales performance vary across regions?
- Which countries are top contributors to overall revenue?

- # Key Features

-  Country-wise sales visualization using maps
-  Bar charts for top-performing countries
-  Interactive slicers for dynamic filtering
-  DAX measures for total sales and performance metrics
-  Cleaned and transformed data using Power Query

-  # Tools & Technologies

- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Power Query  
- Excel (data source)
